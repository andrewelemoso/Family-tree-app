## üÜò Troubleshooting

### Issue: Site not showing after GitHub Pages enabled

**Checklist:**
- [ ] Waited at least 3-5 minutes after clicking Save?
- [ ] Hard refreshed browser? (Cmd+Shift+R on Mac)
- [ ] Tried incognito/private window?
- [ ] Checked if Settings still shows `main` branch and `/ (root)` folder?
- [ ] Looked at Actions tab for deployment errors?

**Solution:**
1. Wait another 5 minutes
2. Hard refresh browser
3. Check Actions tab for errors
4. Verify Settings still configured correctly

### Issue: Tree not rendering

**Checklist:**
- [ ] Opened browser console? (F12)
- [ ] Looking for red error messages?
- [ ] File loaded successfully (check Network tab)?
- [ ] JSON file is valid?

**Solution:**
1. Check browser console for specific errors
2. Hard refresh page
3. Try different browser
4. Check that all files were pushed to GitHub

### Issue: Search not working

**Checklist:**
- [ ] Typed in search box?
- [ ] Searched for names from README?
- [ ] Checked browser console for errors?

**Solution:**
1. Check console for errors
2. Try searching exact names
3. Refresh page and try again

### Issue: Mobile view looks wrong

**Checklist:**
- [ ] Viewport meta tag in HTML (should be there)?
- [ ] CSS media queries working?
- [ ] Trying on actual mobile device?

**Solution:**
1. Check CSS file for media queries (should be there)
2. Hard refresh on mobile
3. Try landscape and portrait modes
4. Test in different browsers
